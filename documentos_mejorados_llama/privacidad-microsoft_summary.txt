Título: Declaración de Privacidad de Microsoft

Fecha de última actualización: enero de 2025

Propósito: Microsoft proporciona una declaración de privacidad para informar a los usuarios sobre la recopilación, uso y protección de sus datos personales. La declaración aborda la importancia de la privacidad y la seguridad de los datos, y busca brindar transparencia sobre las prácticas de Microsoft.

Resumen de contenido:

1. Introducción
- Microsoft reconoce la importancia de la privacidad y la seguridad de los datos personales.
- La declaración de privacidad se aplica a todos los productos y servicios de Microsoft.

2. Datos personales que recopilamos
- Microsoft recopila varios tipos de datos personales, como nombre y dirección de correo electrónico, información de pago, y datos sobre el uso de productos y servicios.
- Los datos pueden ser recopilados directamente de los usuarios o a través de fuentes de terceros.

3. Cómo usamos los datos personales
- Microsoft utiliza los datos personales para proporcionar productos y servicios, mejorar la experiencia del usuario, y realizar publicidad personalizada.
- Los datos pueden ser compartidos con terceros para proporcionar servicios y funcionalidades.

4. Razones por las que compartimos datos personales
- Microsoft comparte datos personales con terceros para proporcionar servicios y funcionalidades, y para cumplir con las leyes y regulaciones aplicables.

5. Cómo acceder a tus datos personales y controlarlos
- Los usuarios pueden acceder a sus datos personales y controlarlos a través de herramientas y configuraciones proporcionadas por Microsoft.

6. Cookies y tecnologías similares
- Microsoft utiliza cookies y tecnologías similares para recopilar datos sobre el uso de los productos y servicios, y para mejorar la experiencia del usuario.

7. Productos proporcionados por tu organización
- Si un usuario utiliza un producto de Microsoft proporcionado por su organización, la organización puede tener acceso a los datos del usuario.

8. Cuenta de Microsoft
- La cuenta de Microsoft permite a los usuarios acceder a productos y servicios de Microsoft, y proporciona una forma de administrar los datos personales.

9. Recopilación de datos de menores
- Microsoft recopila datos de menores de 13 años o según especifique la ley de la jurisdicción, con el consentimiento de los padres o tutores.

10. Otra información importante sobre privacidad
- Microsoft proporciona información adicional sobre la privacidad, como la seguridad de los datos personales y la conservación de los datos.

11. Privacidad de datos de los estados de los Estados Unidos
- Microsoft se adhiere a las leyes de privacidad de datos de los estados de los Estados Unidos, y proporciona información sobre los derechos de los usuarios.

12. Procesamiento de la información personal
- Microsoft proporciona información sobre el procesamiento de la información personal, incluyendo la recopilación, uso y divulgación de datos.

13. Grupos que controlan la recopilación de datos personales
- Microsoft permite a terceros controlar la recopilación de datos personales en ciertas situaciones.

14. Publicidad
- Microsoft utiliza datos personales para proporcionar publicidad personalizada, y proporciona información sobre cómo los usuarios pueden controlar la publicidad.

15. Tecnologías de reconocimiento de voz
- Microsoft utiliza tecnologías de reconocimiento de voz para proporcionar funcionalidades de voz en productos y servicios.

16. Versiones gratuitas o preliminares
- Microsoft ofrece versiones gratuitas o preliminares de productos y servicios, que pueden recopilar datos adicionales.

17. Cambios en esta declaración de privacidad
- Microsoft puede actualizar la declaración de privacidad para reflejar cambios en las prácticas de privacidad.

18. Cómo comunicarse con Microsoft
- Los usuarios pueden comunicarse con Microsoft para obtener más información sobre la privacidad y la seguridad de los datos personales.

19. Capacidades de inteligencia artificial y Microsoft Copilot
- Microsoft proporciona información sobre las capacidades de inteligencia artificial y Microsoft Copilot, y cómo se utilizan los datos personales.

20. Productos para empresas y desarrolladores
- Microsoft proporciona información sobre los productos para empresas y desarrolladores, y cómo se utilizan los datos personales.

21. Productos de productividad y comunicaciones
- Microsoft proporciona información sobre los productos de productividad y comunicaciones, y cómo se utilizan los datos personales.

22. Familia de Microsoft
- Microsoft proporciona información sobre la familia de Microsoft, y cómo se utilizan los datos personales.
